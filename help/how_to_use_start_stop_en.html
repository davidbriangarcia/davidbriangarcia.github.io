How to Start, Stop, and Pause a Macro

Once the macro has been recorded, you can start it with the Play button or save it and assign a key or key combination to it in the settings window for better control.

To stop a macro, you can use the Stop button in the macro window or the Escape button. To pause a macro, you can use the Pause button, which will temporarily halt the execution of the macro until you resume it.

Once the macro start a little window in the top left corner shows with some helper buttons like Stop, Pause/Resume and close. You can disable this windows in the configuration.
