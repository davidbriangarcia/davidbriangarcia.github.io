How to Start, Stop, and Pause a Macro

Starting a Macro

Once you have recorded a macro, you have several ways to start it:

  • Play Button: Click the "Play" button in the main window.
  • Hotkeys: Assign a custom hotkey (e.g., F9) in the settings for quick execution.

Stopping and Pausing a Macro

During macro execution, you can control its flow:

  • Stop: Press the "Stop" button in the control window or use the default Escape key.
  • Pause/Resume: Use the "Pause" button to temporarily halt the macro. Click it again to resume.

The Macro Control Window

When a macro is running, a small control window appears in the top-left corner of your screen. This window provides quick access to:

  • Stop: Immediately terminates the macro.
  • Pause/Resume: Pauses or continues the macro.
  • Close: Closes the control window (the macro will continue to run).

You can disable this control window in the application's configuration settings if you prefer a less intrusive experience.
