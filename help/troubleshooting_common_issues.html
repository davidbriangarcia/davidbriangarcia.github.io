Troubleshooting Common Macro Issues

I can't edit an action in the macro list.

Sometimes, a recorded action can become corrupted or not register correctly, which prevents editing.

Solution: Select the problematic action and use the "Clean Action" feature to remove it. Afterward, you can try recording that part of the sequence again.

I'm having trouble logging in.

Login issues are typically caused by incorrect credentials or a connection problem.

Solution:

  1. Verify that your username and password are correct.
  2. Ensure you have an active internet connection.
  3. If the problem persists, use the password reset option on the website.

Premium features are locked after payment.

Sometimes, the application needs to be restarted for subscription changes to apply correctly.

Solution: Close the application completely and reopen it. This should sync your subscription status and unlock all PRO features. If the issue continues, please contact support.

Solución de Problemas Comunes

No puedo editar una acción en la lista de macros.

A veces, una acción grabada puede corromperse o no registrarse correctamente, lo que impide su edición.

Solución: Selecciona la acción problemática y utiliza la función "Limpiar Acción" para eliminarla. Después, puedes intentar grabar esa parte de la secuencia de nuevo.

Tengo problemas para iniciar sesión.

Los problemas de inicio de sesión suelen deberse a credenciales incorrectas o a un problema de conexión.

Solución:

  1. Verifica que tu nombre de usuario y contraseña son correctos.
  2. Asegúrate de tener una conexión a internet activa.
  3. Si el problema persiste, utiliza la opción de restablecer contraseña en el sitio web.

Las funciones Premium están bloqueadas después de pagar.

A veces, la aplicación necesita reiniciarse para que los cambios de la suscripción se apliquen correctamente.

Solución: Cierra la aplicación por completo y vuelve a abrirla. Esto debería sincronizar tu estado de suscripción y desbloquear todas las funciones PRO. Si el problema continúa, contacta con soporte.
