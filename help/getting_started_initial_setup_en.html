Initial Setup

Goblal Hotkeys:

Allows you to choose a button (or combination of buttons) to start and stop macro recordings.

Ammount of buttons:

You can choose the number of side buttons, their names, and the file location.

Widht of the sidebar:

In this section, you can assign the width of the buttons on the left sidebar.

Helper Windows:

Disabling this option prevents windows from being created in the upper left corner when recording or playing back macros.

Theme:

You can select the theme of the application; the default is dark mode.
