How to Start, Stop, and Pause a Macro

Starting a Macro

Once you have recorded a macro, you have several ways to start it:

  • Play Button: Click the "Play" button in the main window.
  • Hotkeys: Assign a custom hotkey (e.g., F9) in the settings for quick execution.

Stopping and Pausing a Macro

During macro execution, you can control its flow:

  • Stop: Press the "Stop" button in the control window or use the default Escape key.
  • Pause/Resume: Use the "Pause" button to temporarily halt the macro. Click it again to resume.

The Macro Control Window

When a macro is running, a small control window appears in the top-left corner of your screen. This window provides quick access to:

  • Stop: Immediately terminates the macro.
  • Pause/Resume: Pauses or continues the macro.
  • Close: Closes the control window (the macro will continue to run).

You can disable this control window in the application's configuration settings if you prefer a less intrusive experience.

Cómo Iniciar, Detener y Pausar un Macro

Iniciar un Macro

Una vez que hayas grabado un macro, tienes varias formas de iniciarlo:

  • Botón de Reproducir: Haz clic en el botón "Reproducir" en la ventana principal.
  • Teclas de Acceso Rápido: Asigna una tecla de acceso rápido personalizada (por ejemplo, F9) en la configuración para una ejecución rápida.

Detener y Pausar un Macro

Durante la ejecución del macro, puedes controlar su flujo:

  • Detener: Presiona el botón "Detener" en la ventana de control o usa la tecla Escape por defecto.
  • Pausar/Reanudar: Usa el botón "Pausar" para detener temporalmente el macro. Haz clic de nuevo para reanudarlo.

La Ventana de Control del Macro

Cuando un macro está en ejecución, aparece una pequeña ventana de control en la esquina superior izquierda de tu pantalla. Esta ventana proporciona acceso rápido a:

  • Detener: Termina inmediatamente el macro.
  • Pausar/Reanudar: Pausa o continúa el macro.
  • Cerrar: Cierra la ventana de control (el macro continuará ejecutándose).

Puedes desactivar esta ventana de control en la configuración de la aplicación si prefieres una experiencia menos intrusiva.
