Getting Started: Initial Setup

Step 1: Download and Run

  1. Download the application from the official website.
  2. AutoPlay is portable, so there's no installation needed. Simply run the .exe file.
  3. Note: Windows may show a SmartScreen warning because it's a new application. If this happens, click "More info" and then "Run anyway".

Step 2: Understanding the Configuration

Before you start recording, it's a good idea to visit the Config window to set up AutoPlay to your liking. Here’s what you can customize:

  • Global Hotkeys: Choose a key or key combination to globally start and stop macro recordings from any application.
  • Sidebar Buttons: Customize the number of quick-access buttons on the sidebar, their names, and where their macro files are located.
  • Sidebar Width: Adjust the width of the sidebar to fit your screen and preferences.
  • Helper Windows: You can disable the small windows that appear in the corner of your screen when recording or playing back macros.
  • Theme: Choose between a light or dark theme for the application.

Step 3: What's Next?

Once you're all set up, you're ready to create your first macro. Continue to the next guide to learn how.

Primeros Pasos: Configuración Inicial

Paso 1: Descargar y Ejecutar

  1. Descarga la aplicación desde el sitio web oficial.
  2. AutoPlay es portable, por lo que no necesita instalación. Simplemente ejecuta el archivo .exe.
  3. Nota: Es posible que Windows muestre una advertencia de SmartScreen porque es una aplicación nueva. Si esto ocurre, haz clic en "Más información" y luego en "Ejecutar de todas formas".

Paso 2: Entendiendo la Configuración

Antes de empezar a grabar, es buena idea visitar la ventana de Configuración para ajustar AutoPlay a tu gusto. Esto es lo que puedes personalizar:

  • Atajos Globales: Elige una tecla o combinación de teclas para iniciar y detener globalmente las grabaciones de macros desde cualquier aplicación.
  • Botones Laterales: Personaliza el número de botones de acceso rápido en la barra lateral, sus nombres y dónde se encuentran los archivos de sus macros.
  • Ancho de la Barra Lateral: Ajusta el ancho de la barra lateral para que se adapte a tu pantalla y preferencias.
  • Ventanas de Ayuda: Puedes desactivar las pequeñas ventanas que aparecen en la esquina de tu pantalla al grabar o reproducir macros.
  • Tema: Elige entre un tema claro u oscuro para la aplicación.
